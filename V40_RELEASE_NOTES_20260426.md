# v40 修正说明 - 2026-04-26

本版只集中修复用户指出的模型问题。

## 修复内容

1. 标准人体模型改为内嵌加载：不再依赖运行时 fetch 标准人体 JSON，避免离线打开、file:// 限制、缓存未刷新时回退到旧程序化模型。
2. 模型界面彻底去掉 6 个指标浮层：体重、BMI、腰臀比、目标热量、蛋白、水分不再显示在模型画布上，也不会遮挡营养环。
3. 身体维度塑形加强：胸、腰、腹、臀、上臂、前臂、大腿、小腿、身高、体重、体脂和照片捏形参数都会影响标准人体网格。
4. 保留营养环：营养环仍可显示摄入进度，但不再被指标卡片遮挡。
5. 保持历史数据结构：身体维度和摄入仍按本地自然日 0:00-23:59 记录，并保留 JSON / CSV 导出。
